# Customer Retention & Churn Analysis

## 📊 Overview
This project analyzes customer churn and retention patterns in a subscription-based business. The goal is to identify key factors that lead to customer loss and provide insights to improve retention.

## 🎯 Objectives
- Analyze churn rates across different plans
- Identify key churn reasons
- Perform retention and cohort analysis
- Evaluate customer lifetime value (CLV)

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Power BI (Dashboard)

## 📈 Key Insights
- Higher churn observed in Basic plans compared to Premium
- Customer engagement strongly impacts retention
- Retention decreases gradually over time
- Pricing and feature usage are key churn drivers
