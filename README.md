# Customer Retention & Churn Analysis

## 📊 Project Overview
This project analyzes customer churn and retention patterns in a subscription-based business using Python and Power BI.

The goal is to identify why customers churn and provide insights to improve retention and customer lifetime value.

---

## 🎯 Objectives
- Understand customer churn behavior
- Identify key factors driving churn
- Analyze retention trends over time
- Evaluate customer lifetime value (CLV)

---

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Power BI (Dashboard & Visualization)

---

## 📈 Key Insights
- Higher churn observed in lower-tier (Basic) plans
- Customers with low engagement are more likely to churn
- Retention decreases over time (cohort trend)
- Pricing and feature usage are key churn drivers
